📝 To-Do App (Android)

A simple and clean To-Do List Android application built as a campus project. This app allows users to manage daily tasks efficiently with an intuitive interface and lightweight performance.

🚀 Features
✅ Add new tasks
✏️ Edit existing tasks
🗑️ Delete tasks
📋 View all tasks in a structured list
✔️ Mark tasks as completed (if implemented)
💾 Persistent data storage (based on your implementation — e.g., local database or shared preferences)
🛠️ Tech Stack

This project is built using:

Kotlin – Primary programming language
Android SDK – Mobile app development framework
Gradle (Kotlin DSL) – Build system (build.gradle.kts)
Android Studio – Recommended IDE

Room Database / SQLite
ViewModel & LiveData (MVVM Architecture)
RecyclerView for task listing
